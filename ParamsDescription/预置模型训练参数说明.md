# 预置模型运行常用参数说明

更新时间： 2018-12-22

- Mxnet-moxing运行参数

  | 参数               | 功能                                                      | 默认值   | 示例                                     | 必要性 |
  | ------------------ | --------------------------------------------------------- | -------- | ---------------------------------------- | ------ |
  | num_classes        | 数据的类别数                                              | 无       | 如数字识别就是0~9一共10类 num_classes=10 | 必填   |
  | layer_before_fullc | 记录全连接层前的later名称，用来适配网络输出（已自动添加） | flatten0 | resnet50是layer_before_fullc = flatten0  | 必填   |
  | batch_size         | 每次更新模型输入的数据数量，(过大会造成显存不足）         | 32       | batch_size=32                            | 选填   |
  | num_epoch          | 所有的数据经过一次模型为一个epoch，控制训练的epoch数      | 10       | num_epoch=10                             | 选填   |
  | lr                 | 学习率，每次更新模型时的步长                              | 0.1      | lr=0.1                                   | 选填   |
  | split_spec         | 划分数据集为训练集和预测集的比例                          | 0.8      | split_spec=0.8                           | 选填   |

- TensorFlow-moxing运行参数

  | 参数               | 功能                                                      | 默认值   | 示例                                     | 必要性 |
  | ------------------ | --------------------------------------------------------- | -------- | ---------------------------------------- | ------ |
  | batch_size         | 每次更新模型输入的数据数量，(过大会造成显存不足）         | 32       | batch_size=32                            | 选填   |
  | eval_batch_size    | 每次验证模型输入的数据数量，(过大会造成显存不足）         | 1        | batch_size=1                            | 选填   |
  | max_epoches          | 所有的数据经过一次模型为一个epoch，控制训练的epoch数      | 100      | max_epoches=100                             | 选填   |
  | learning_rate_strategy | 学习率，每次更新模型时的步长                        | 0.002      | learning_rate_strategy=0.002                                   | 选填   |
  | split_spec         | 划分数据集为训练集和预测集的比例                          | 0.8      | split_spec='train:0.8,eval:0.2'                           | 选填   |
  | max_training_time | 最大训练时间（小时），超时则训练退出 | 10000    | max_training_time=10000  | 选填   |
